Leaflet.Draw 的插件，允许在 Leaflet 地图上添加/移动/删除自定义 SVG 文件。 它允许将 SVG 定义为 XML 文本，并将其转换为可以由 Leaflet 显示为可编辑功能的 SVG 对象。
